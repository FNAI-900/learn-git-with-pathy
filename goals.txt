علمت أساسيات الأدوات اللي بنستخدمها مثل Git و VS Code وكيفية تثبيتها وتجهيز Git للاستخدام. فهمت إيش هو Version Control وليش نحتاجه، والفرق بين Git و GitHub وأهميتهم للمطورين.

كذلك تعلمت استخدام الـ Command Line للتنقل بين الملفات والمجلدات وإنشائها وتنظيمها، وفهمت الفرق بين المسارات المطلقة والنسبية. وأخيرًا تعلمت إنشاء مشروع باستخدام Git، تتبع وحفظ التغييرات، وربط المشروع بـ GitHub ورفعه عليه.
